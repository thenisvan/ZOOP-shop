# ConsoleShop

## What's a console shop?
- A console application where you can `register` as a user and add items as the `owner` of the shop.
  - Basically simulating an **online shop.**
- As the `owner` you can do the following:
  - Add a item.
  - View transactions.
  - View buyers details.
  - View your items.
  - Remove a buyer.
  - Your **_added products_**,**_transactions_**, and **_customers_** are stored into a **CSV** file so that when you exit the program, you can **still see those informations.**
- And the `buyers` can do the following:
  - Cash in.
  - Go shopping. (can add to cart or buy)
  - View Info.
  - View balance.
  - View your cart.
  - Clear your cart.
  - Checkout.
  - View your bought items.
  - Customers **_money_**, **_cart_** , and **_bought products_** are also stored into a **CSV** file for later use.

## Steps to get started

- Clone the **Repository**

- Open it in your ***favorite editor***

- Run **Main.java**

- Login using this:

    - `username:` **admin**

    - `password:` **pass**

- You can now add items!

- Register buyers and test it!
